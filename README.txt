Ce programme python permet la création de listes de caractères ("wordlist") qui sont rédigées et stockées sous la forme
d'un ficher texte (.txt). Ici, il permet l'obtention d'une wordlist de chaînes de 5 caractères.
Ces listes sont utilisées notamment pour cracker des mots de passe en essayant les différentes combinaisons de caractères 
possibles pour une longueur donnée.

Cependant, ce programme reste gourmand en ressources de disque c'est la raison pour laquelle il est préférable de ne pas
l'exécuter sur son ordinateur personel avec une longueur donnée trop importante.
